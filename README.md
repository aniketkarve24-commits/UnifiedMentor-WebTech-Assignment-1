🌐 Unified Mentor - Cyber Sequerity Intern.

This repository contains the assignments and projects completed during my Cybersecurity & Web Development Internship at Unified Mentor. The focus of this module was to master the fundamentals of web architecture, backend development, and security auditing.

👤 Intern Information

Domain: Cybersecurity 

Organization: Unified Mentor

Assignment: #1 - Web Technologies Fundamentals

🚀 Project Overview

The goal of this project was to bridge the gap between building web applications and understanding their security implications. It covers everything from basic frontend design to advanced backend integration and traffic interception using industry-standard tools.

🛠 Tech Stack & Tools

Frontend: HTML5, CSS3

Backend: PHP 8.x

Database: MySQL

Server Environment: XAMPP (Apache & MySQL)

Security Tools: Burp Suite Professional, Wappalyzer, Browser Developer Tools

📂 Assignment Tasks

1. Basic HTML & Form Handling

Created 5 interactive HTML pages with various forms, tables, and lists.

Practiced data submission using GET and POST methods.

Performed real-time DOM manipulation using browser developer tools.

2. PHP & Database Integration

Set up a local development environment using XAMPP.

Developed 10+ PHP scripts covering loops, functions, and global variables.

Built a Secure Login System connected to a MySQL database.

3. Web Request/Response Analysis (Security)

Used Burp Suite to intercept and analyze HTTP/HTTPS traffic.

Modified parameters in real-time to observe server behavior.

Analyzed HTTP headers (User-Agent, Cookies, etc.) for security auditing.

4. Source Code & Technology Analysis

Audited 10+ websites to identify their tech stack using Wappalyzer.

Inspected source code for sensitive information leaks (hidden fields, developer comments).

Analyzed client-side vs. server-side validation logic.
